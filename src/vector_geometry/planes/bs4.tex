Find all unit vectors parallel to the plane $x + 2y + 3z = 5$ and the $xy$-plane.
\end{Exercise}
\begin{Answer}
$\pm \left(\frac{2}{\sqrt{5}},\;\frac{-1}{\sqrt{5}},\;0\right)$.
