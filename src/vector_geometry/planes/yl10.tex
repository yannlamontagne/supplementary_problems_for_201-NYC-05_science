Given the plane $\mathcal{P}:\;3x+2y+z=6$.
\Question Sketch the given plane using the $x$, $y$ and $z$-intercepts.
\Question Using projection(s) find the distance between the origin and $\mathcal{P}$.
\Question Find the angle between $\mathcal{P}$ and the $xz$-plane.

\end{Exercise}
