\begin{Exercise}[
name={},
title={}, 
difficulty=0,
origin={\cite{BS}}]
Determine whether the point $(1,\; 2,\; 1)$ is between planes $6x - 3y + 6z = -3$ and $4x - 2y + 4z = 2$.
\end{Exercise}
\begin{Answer}
The point is not in between the planes.
\end{Answer}
