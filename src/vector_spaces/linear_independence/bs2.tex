Find all values of $t$ such that $\set{(1,\; -t,\; 2),\; (t,\; 3,\; -1),\; (3t,\; 5,\; -4)}$ is a linearly independent set of vectors in $\R^3$.
\end{Exercise}

$\set{t\in\R\suchthat t\neq-1\text{ and }t\neq-7}$
