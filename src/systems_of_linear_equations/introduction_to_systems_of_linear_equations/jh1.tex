\begin{Exercise}[
name={},
title={}, 
difficulty=0,
origin={\cite{JH}}]
Is there a two-unknowns
linear system whose solution set is all of \( \Re^2 \)?
\end{Exercise}

\begin{Answer}
Yes, this one-equation system:
\begin{equation*}
0x+0y=0
\end{equation*}
is satisfied by every \( (x,y)\in\Re^2 \). 
\end{Answer}
