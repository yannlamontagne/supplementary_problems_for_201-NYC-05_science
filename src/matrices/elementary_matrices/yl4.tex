\begin{Exercise}[
name={},
title={}, 
difficulty=0,
origin={\cite{YL}}]
Prove: If $E_i$ are elementary matrices and $E_n\cdots E_2E_1A$ is invertible then $A$ is invertible.
\end{Exercise}

\begin{Answer}
Hint: A matrix is invertible if and only if it can be expressed as a product of elementary matrices.
\end{Answer}
